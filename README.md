# CassetteTapeDump
Output all cassette tapes on your server to ogg files.

## How to use
Open the F1 console with admin and enter the "tapedump" command!

管理者で、F1コンソール開いてtapedumpってうって。
RUSTサーバ直下に数字名のoggファイルが作成されるよ。
